# Critique Checklist

A systematic professional critique framework you can apply to any piece.

Use this checklist on the user's art (especially shields, sprites, and icons) before giving advice.

## Evaluation Order (always follow this sequence)
1. Silhouette & Readability at target size(s)
2. Edge cleanliness and pixel consistency
3. Value structure and contrast hierarchy
4. Form and depth (shading logic)
5. Color harmony and material clarity
6. Center focal elements and details
7. Animation / multi-frame considerations (if applicable)
8. Production constraints (palette size, scale targets, etc.)

Never skip steps. Never start with polish when fundamentals are broken.
