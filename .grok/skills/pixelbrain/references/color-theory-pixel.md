# Color Theory for Pixel Art

Building and using limited palettes that actually work at small sizes.

Pixel art color is brutally constrained. A palette that looks good on a large canvas can die at 16x16 or in a game UI.

## Topics
- Choosing hues that survive dithering and scaling
- Temperature, value, and chroma control in tiny palettes
- Harmonies that read on limited hardware / old monitors
- Palette cycling, ramps, and shared colors across a set
- Common palette crimes and how to avoid them

*(Reference palettes, decision frameworks, and case studies to be added over time.)*
