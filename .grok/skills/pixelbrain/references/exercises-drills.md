# Exercises & Drills

Progressive practice drills from absolute beginner to advanced mastery.

Deliberate practice is how you actually ascend. Random doodling is not enough.

## Structure
- Novice fundamentals drills (silhouette, 1-bit readability, clean lines)
- Intermediate form & material drills
- Advanced limited-palette studies and animation loops
- Targeted "fix the weakest part" drills using the user's own pieces (e.g. Void Shield center focal work)
- Time-boxed challenges (20-minute sessions)

Each drill includes: goal, constraints, success criteria, and common failure modes.
