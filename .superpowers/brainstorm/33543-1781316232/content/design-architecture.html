New Files (6)

codex/core/pixelbrain/character-bytecode-compiler.js CHARACTER-SPEC-v1 → flat uniform object. Pure, deterministic, browser-safe.
src/pages/Combat/shaders/character.frag.glsl GLSL ES 1.0 fragment shader. SDF body + noise cloth + school glow.
src/pages/Combat/scenes/CharacterShaderRenderer.js Phaser scene utility. Creates Shader GO, bakes to DynamicTexture.
src/lib/character-enhancement.js snapshot → /api/character/enhance → parsed JSON uniforms.
server/routes/character-enhance.js Backend proxy: Anthropic SDK → Claude Haiku (key stays server-side).
tests/core/pixelbrain/character-bytecode-compiler.test.js Determinism + schema contract tests for the compiler.

Modified Files (2)

src/pages/Combat/assets/combatAssets.js buildScholarTexture / buildWraithTexture replaced by CharacterShaderRenderer calls.
src/pages/internal/pixel-lotus/ActorForgeLab.tsx Adds "Forge & Enhance" button, shows AI-enhanced preview alongside raw forge.
