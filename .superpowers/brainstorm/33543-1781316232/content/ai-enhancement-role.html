A — AI as Bytecode Generator

Describe a character in plain English. Claude (via the API) interprets it and emits the full CHARACTER-SPEC-v1 + shader uniform overrides — SDF distortion amount, noise texture scale, glow intensity. You author characters by describing them, not by tweaking sliders.

→ AI →
AI analyzes rendered frame → proposes shader uniform adjustments

B — AI as Visual Enhancer

The shader renders the character from bytecode. Claude then analyzes the output and suggests refinements — better contrast, additional detail layers, glow adjustments — which are fed back as updated shader uniforms. Iterative visual polish loop driven by AI critique.

Full pipeline: NL description → spec → GPU render → AI enhancement pass

C — Both: Generation + Enhancement

AI acts at both ends. Input side: describe your character in natural language, Claude emits the mathematical bytecode (CHARACTER-SPEC-v1 + shader uniforms). Output side: after the GPU renders, an AI enhancement pass proposes a second set of shader overlays (extra detail, atmosphere, school-specific effects) applied in a second draw pass.
