Components — What Gets Built

File Status Responsibility
codex/core/pixelbrain/cell-boundary-tracer.js NEW Cell occupancy set → ordered boundary polygon vertices. Catmull-Rom smoothing to eliminate staircase jaggies. Per-part isolation (traces each body region separately).
codex/core/pixelbrain/svg-path-builder.js NEW Polygon vertices → SVG d attribute string. Handles closed paths, smooth curves, stroke vs fill mode. School outline color via darkened fill anchor.
codex/core/pixelbrain/character-to-svg.js NEW Full character SVG assembly. Takes region-fill-amp output, groups by partId, stacks fills → shade zones → ink outline. Returns <svg> string with CSS classes for school theming.
codex/core/pixelbrain/renderer-registry.js NEW Registry mapping renderer name → render function. 'pixelart' (current) and 'illustrated' (new). Pluggable — future renderers register here.
codex/core/pixelbrain/character-foundry.js MODIFY Accept { renderer: 'illustrated' } option. Route to SVG path when requested. Default unchanged → existing PNG output unaffected.
src/pages/Combat/scenes/CharacterShaderRenderer.js MODIFY bake() detects SVG output → encodes as data:image/svg+xml;base64,… → loads via existing addBase64 path. Optional: load at combat scale (64×96) vs portrait scale (512×768).
src/pages/internal/pixel-lotus/ActorForgeLab.tsx MODIFY Before/after panel shows SVG preview inline (<img src="data:image/svg+xml;…">). Infinite resolution — no PNG decode artifacts in the preview.

All existing pixel art tests remain green. SVG renderer is an additive path.

School variation is zero-cost — the material registry already parameterizes fills per material. Point a school's character profile at its material anchors and the SVG inherits the right colors.

The CSS Superpower

Because the SVG emits semantic CSS classes (.pb-part-robe, .pb-part-skin, .school-void), school theming works without a single re-render:

.school-void .pb-part-robe  { fill: var(--void-robe-color);  }
.school-void .pb-outline    { stroke: var(--void-ink-color); }
.school-psychic .pb-part-robe { fill: var(--psychic-robe-color); }
