Which Final Fantasy style?

Each era has a distinct approach to pixel art. The current system is blocky because every shape is built from rectangular cell grids — we'd be replacing that with hand-crafted outline + fill rendering. Which of these resonates?

A — FF6 Battle Sprite

SNES era. ~24px tall at 1×, bold 1px outline, flat colour fills with dithered shading. Compact and readable. What most people picture when they say "Final Fantasy pixel art". Fast to render, very legible at small sizes in combat.

B — FF Tactics / Vagrant Story

PSX era. Taller and slimmer proportions, visible tailoring detail on robes and coats, personality-first design. Works beautifully isometric. Higher pixel budget per character but reads as "scholarly" more naturally.

C — Octopath / HD-2D

Switch era. Larger 32×32 canvas (128px at 4×), strong depth shading, elaborate clothing layers, expressive face with visible highlights. Each character reads as a fully realised person. Most work per character, most visual impact.

All three replace the current blocky cell-grid system with a proper outline + fill + shading approach. The current system rasterizes rectangular cells at 4× — no curves, no anti-aliasing, no personality. Any of these would be a fundamental improvement. The difference is proportions, canvas size, and detail level.
