# Competitor Delta Analysis

## Principle
Competitive analysis requires comparing a video's SEO package against category
benchmarks. The delta between a video's scores and category averages reveals
relative strengths and weaknesses.

## Deterministic Checks
- Category benchmark comparison (deferred: requires multi-video dataset)
- Relative positioning within niche (deferred)

## Failure Modes
- Insufficient comparison data (deferred)

## Critique Language
| Flag | Language |
|------|----------|
| COMPETITOR_DATA_INSUFFICIENT | Competitor delta analysis requires a reference dataset. This feature is deferred to a future release. |

## Scoring Impact
Competitor delta analysis is deferred. Future implementation will add a comparative
layer to the critique report, benchmarking against category averages and top
performers in the same niche.
