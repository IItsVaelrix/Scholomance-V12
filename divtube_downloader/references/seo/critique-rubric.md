# Critique Rubric

## Principle
The critique rubric translates raw engine scores into strategist-grade recommendations.
Every recommendation must trace to a specific flag and reference principle. The rubric
ensures consistency across reports regardless of whether prose is generated by LLM or
template.

## Deterministic Checks
- Overall score is weighted average of four engines (thumbnail, title, tag, performance)
- Replication value derived from the same four scores
- Every flag maps to a reference block's critique language
- Performance confidence guard adjusts interpretation of telemetry data

## Failure Modes
- Missing thumbnail: thumbnail score is null, report notes data gap
- Missing telemetry: performance score degraded, confidence set to LOW
- Missing tags: tag score reflects empty set, coverage is zero

## Critique Language
| Flag | Language |
|------|----------|
| OVERALL_CRITICAL | The video package has critical weaknesses across multiple dimensions. Immediate revision is recommended before further promotion. |
| OVERALL_STRONG | The video package is well-optimized across thumbnail, title, tags, and performance. Minor refinements may improve marginal metrics. |

## Scoring Impact
The rubric governs the final report structure. Scores determine the verdict, the
prioritized action list, and the rewrite suggestions. The rubric ensures that
every critique line is evidence-based and actionable.
